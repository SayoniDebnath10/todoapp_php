# To-Do App (PHP + MySQL)

A traditional full-stack To-Do application built using PHP, MySQL, HTML and CSS.

**Current Version:** v1.0

## Features
- Create (Add) Tasks
- Read (View) Tasks
- Update (Edit) Tasks
- Delete Tasks
- Server-side CRUD Operations
- MySQL Database Integration

## Tech Stack
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Git
- GitHub

## Database
**Database Name:** `todolist`

**Table:** `tasks`

| Column | Type |
|---------|------|
| task_id | INT (Primary Key, Auto Increment) |
| task_name | VARCHAR(255) |
